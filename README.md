# official-footballwager-pages

本仓库用于归档和发布多个独立 HTML 页面。这些页面是项目中使用的静态资源，不针对任何单个域名或具体网站。

## 项目简介

该仓库集中存放与足球竞猜相关的独立页面。每个页面均为完整的 HTML 文档，可独立运行或嵌入到其他环境中。仓库的主要目的是方便版本管理、历史回溯与静态部署。

## 目录说明

```
.
├── pages/           # 存放所有独立 HTML 页面
├── assets/          # 公共资源（CSS、JS、图片等）
├── README.md        # 本文件
└── .gitignore       # Git 忽略规则
```

- `pages/`：每个子目录或文件对应一个独立功能页面，命名清晰，便于识别。
- `assets/`：存放多个页面共用的静态资源，避免重复。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接通过浏览器打开。
- 页面之间无强耦合，可按需引用或单独部署。
- 归档时建议保留原始目录结构与资源引用路径，避免依赖失效。

## 维护说明

- 如需新增页面，请在 `pages/` 下创建对应文件夹或文件，并确保资源引用正确。
- 修改公共资源时，请检查所有引用该资源的页面，避免出现显示异常。
- 建议每次提交前对页面进行基本功能测试，确保无明显错误。
- 仓库不包含任何外部服务配置或后端逻辑，仅作为前端静态资源存档。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/official-footballwager-pages.git
   ```
2. 直接打开 `pages/` 下的 HTML 文件即可预览。
3. 也可将整个仓库部署到任意静态托管服务（如 GitHub Pages、Netlify 等）。

## 注意事项

- 本仓库不包含任何用户数据、后端接口或动态内容。
- 页面内容仅供归档与展示使用，不构成任何形式的服务承诺。
- 请勿在本仓库中存放敏感信息或未经授权的资源。
